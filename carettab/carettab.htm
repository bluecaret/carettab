Time Settings

Display Primary Clock

OFF

Display Seconds

If enabled, displays the seconds on the primary time (applies only to the 'primary clock').
OFF (12:30)
ON (12:30:40)

Display Meridiem

Toggle the meridiem (am/pm) on/off (only applies to the 'primary clock' and if set to 12 hour format).
OFF (12:30)
ON (12:30 AM)

24 Hour Clock

If enabled, displays all clocks in a 24 hour format, i.e. military time (the meridiem will be hidden if this is enabled).
12 Hour
24 hour

OFF (January 31)

If enabled, displays the date in a short format.

ON (01/31/2016)

If enabled, displays the date in the format used by the United States.
January 31, 2000 (01/31/00)

Colors

Set custom background color

Click in the text box below and type in a color code in HEX, or use the popup controls to choose a color.

Choose a color theme

Click on a circle to choose a color theme. The top color represents the background color and the bottom color represents the foreground (text) color.
